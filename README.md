# Files-Manipulation
Files Manipulation in C and Cpp

Cpp:	fstream

- Compile: $ g++ -o Fstream Fstream.cpp

- Run: 	   $ ./Fstream

C: 	FILE*

With a thread runing copy content of file to another

- compile: $ gcc -o filecopy filecopy.c -lpthread

- Run:     $ ./filecopy
